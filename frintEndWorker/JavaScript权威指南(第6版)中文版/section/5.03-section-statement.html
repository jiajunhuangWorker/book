5.3声明语句
var 和function都是声明语句它们声明或者定义变量或函数.
声明语句本身什么也不做,但它有一个重要的意义，通过创建变量和函数,可以更好地组织代码的语义
5.3.1var
关键字var之后跟随的是要声明的变量列表
如果var语句出现在函数体内,那么它定义的是一个局部变量,其作用域就是这个函数.如果在顶层代码中使用var语句,它声明的是全局变量
var声明的变量无法通过delete删除的
如果var语句中变量没有指定初始化表达式.变量的值初始为undefined
5.3.2function
关键字function用来定义函数
function是要声明的函数的名称的标识符
定义函数时，并不执行函数体内的语句，它和调用函数时待执行的新函数对象相关联
函数声明语句通常出现在javascript代码的最顶层,也可以嵌套在其他函数体内,但在嵌套时,函数声明只能出现在在所嵌套的数的顶部（函数定义不能出现在if和while循环语句或者任何语句中）
由于函数声明位置限制,Ecmascript标准规范并没有将函数声明归类为真正的语句
函数声明放在其他语句内的做法并不具备可移植性,函数声明语句和函数定义表达式包括相同的函数但二者仍然不同
two way create new function object 函数声明语句中的函数名是一个变量名,变量指向函数对象
和var 声明变量一样,函数定义语句中函数被显式地提前到了脚本或者函数的顶部,因此它们在整个脚本和函数内都可见的
使用var的话,只有变量声明提前了,变量初始化代码仍然在原来的位置,然而使用声明语句的话函数名称和函数体均提前
使用函数声明语句的话,函数名称和函数体均提前:脚本中的所有函数和函数中所有嵌套的函数都会再当前上下文中其他代码之前声明
和var 一样函数声明语句创建的变量无法删除,不是只读,变量值可以重写
