5.4条件语句
条件语句是通过判断指定表达式的值来决定执行还是跳过某些语句
这些语句是代码的决策点,有时候称为分支
5.4.1 if
if语句是一种基本的控制语句,它让javascript程序可以选择执行路径,更准备地说,就是有条件地执行语句
if(expression)statement
if(expression)statement1 else statement2
5.4.2else if
if/else语句通过判断一个表达式的计算结果来选择执行两条分支中的一条
5.4.3switch
if语句在程序执行过程中创建一支分支,并且可以使用else if 来处理多条分支.
switch(expression){statements}
1.switch(val){case 1：//执行1 break;case 2：//执行2 break; default: break;}
2.switch(val){case 1:case2: //执行语句 break; case 3:case4: //执行语句 break; default: break;}
switch语句首先计算switch关键字的表达式,然后按照从上到下的顺序计算每个case后的表达式,直到执行到case的表达式的值与switch的表达式的值相等时为止
由于对每个case的匹配操作实际上是恒等运算符比较而不是相等运算符比较,因此,表达式和case的匹配并不会做任何类型转换
