12.0服务端javascript
javascript嵌入文浏览器的原理,并涵盖庞杂的客户端javascript api
JavaScript是基于web的编程语言,因为绝大部分的javascript代码是为web浏览器而编写
Rhino是基于java的javascript解析器,实现了通过javascript程序访问整个java api
node是google 的v8 javascript解析器的一个特别版本,它在底层绑定了Rosix api,包含文件,进行,流和套接字等,并侧重于异步i/o,网络和http
Rhino程序能使用java的swing框架創建圖形ui,而node上運行的JavaScript程序可以像shell驕傲本那樣操作文件
