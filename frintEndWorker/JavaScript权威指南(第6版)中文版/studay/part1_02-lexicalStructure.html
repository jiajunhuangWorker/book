2.1字符集
javascript程序是unicode字符集编写的，unicode是ascll和lain-1超集,几乎支持地球所有的语言
2.1.1区分大小写,严格区分大小写
javascript区分大小写,html不区分大小写,xhtml区分大小写
2.1.2空格，换行符，格式控制符
javascript会忽略程序中标识之间的空格
unicode格式控制字符控制着文本视觉显示
javascript忽略程序中标识token之间空格
空格符(\u0020),水平制表符(\u0009),垂直制表符(\u000B),换页符(\u000c),不中断空白(\u00A0),字符序标记(\ufeff),javscript:换行符（\u000a） 回车符（\u000d）,行分隔符（\u2028）,段分隔符（\u2029）
2.1.3.unicode转义
有些计算机硬件和软件里无法显示或输入Unicode字符全集,javascript定义了特殊的序列6个ascll字符来代替任意16位
2.1.4标准化
unicode 允许使用多种方法对同一个字符进行编码
unicode 和 ascii表示同某一个字符的时候，他们是显示同一个字符但是它们的二进制编码表示不一样
unicode标准为所有字符定义了首选编码格式，给出了标准化处理方式,将文本转换为适合的标准格式

2.3直接量
程序中直接使用的数据值

2.4.标识符和保留字
标识符：
就是一个名字
对变量和函数进行命名或者循环语句中的标识符
example:_,$,字母，数字，数字是不允许开头是数字的
出于移植性和易于书写，我们通常用ascll和数字书写标识符
保留字:
javascript把一些标识符拿出来用做自己的关键字,这些关键字不能被用做标识符
unicode对字符进行分类,这种分类使用通用类别值,Mc,Mn,Pc
Mc:基字符的修改中影响了基字符标志位宽度的间距字符
Mn:基字符的修改中出现的非间距字符
Pc:连接两个字符的连接符或标点符号
2.5.可选的分号
;代表了语句的结束
return，continue，break这三个关键字紧跟着换行,javascript则会在换行处添加分号
example:
return true; 在javaascript被解析成为 return;true;导致出错
x ++ y 在javaascript被解析成为x;++y导致出错
